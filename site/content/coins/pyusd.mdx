## Integration notes

On **Solana**, PYUSD uses **Token-2022** extensions, so wallet and program support varies. Paxos currently publishes canonical PYUSD deployments for **Ethereum, Solana, and Arbitrum**, and PayPal terms also list **Stellar** support, so treat chain-specific addresses and capabilities as part of integration setup rather than assuming one universal token surface.

## Product surface

PayPal/Venmo custody and promotional yield are **off-chain**—on-chain integrations should still assume standard issuer controls.
